# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0-rc.1] - Version 1 Release Candidate
### Added
- Fully integrated Generic Probe Engine, executing live mutations to evaluate entity constraints.
- Formalized 11-category failure taxonomy trapping all arbitrary execution crashes.
- Unsupervised authentication engine utilizing generic HTML5 field heuristic lookups.
- Headless, safety-first topological mapping engine (DiscoveryEngine).
- Deterministic behavioral learning and knowledge diff extraction against runtime evidence.

### Removed
- Unreachable stub modules: `CorrelationEngine`, `CompletenessEvaluator`, `DatabaseObserver`, `ProbeOrchestrationEngine`.
- Speculative architectural documents outside of consolidated execution flows.
